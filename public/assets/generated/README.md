## Delivery Rules

- Use AVIF first, WebP as the fallback, and PNG when alpha fidelity or metadata use requires it.
- All illustration assets have transparent backgrounds and are designed for both the warm limestone light theme and deep energy-green dark theme.
- French interface labels should remain live HTML. Only the Open Graph image contains baked-in French marketing copy.
- Desktop and mobile compositions are separate files where the layout needs a materially different crop.
- Do not recolor the current-green accent. It is reserved for energy flow, anomaly detection, recovered value, and validated outcomes.
