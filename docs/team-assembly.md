# SaveWatt Team Assembly

## Purpose

This document assembles the working teams for SaveWatt so we can execute the landing page program with clear ownership.

Primary goal:

Create, design, build, and launch a premium French landing page for SaveWatt supported by strong brand strategy, differentiated copy, competitor intelligence, SEO structure, and on-brand visual assets.

Current phase:

Phase 1 is a planning swarm only.
This round does not build the website.
This round produces the master plans that the Web Building Team will execute in the next round.

## Team Model

We will operate with four coordinated teams:

1. Copywriting Team
2. Design Team
3. Copywriting + SEO Strategy Team
4. Web Building Team

All four teams are coordinated by one orchestration layer.

## 1. Orchestrator

### Mission

Direct the entire project from planning through launch, maintain sequence, resolve dependency conflicts, and keep every team aligned to one landing page objective.

### Skills

- `paseo-orchestrate`
- `find-skills`
- `design-brief`
- `design-principles`
- `design-critique`

### Responsibilities

- Define project scope and success criteria
- Decide execution order across teams
- Trigger the correct team at the correct time
- Prevent drift between brand, copy, design, SEO, and implementation
- Run review checkpoints before build and before launch

### Inputs

- SaveWatt brochure
- Competitor references
- Brand direction
- Team outputs from each squad

### Outputs

- `docs/vision.md`
- `docs/team-assembly.md`
- review notes
- approval checkpoints

## 2. Copywriting Team

### Mission

Write the landing page messaging platform and conversion copy for SaveWatt in a tone that feels expert, credible, premium, and high-trust for the French energy optimization market.

### Skills

- `copywriting`
- `copywriting-core`
- `marketing-psychology`
- `copy-editing`
- `hero-generator`

### Responsibilities

- Write headline and subheadline options
- Develop CTA language
- Draft all core landing page sections
- Translate service complexity into simple business language
- Build proof-driven persuasion with clarity, not hype
- Keep copy aligned with SaveWatt’s no-risk and success-fee model

### Inputs

- Brand strategy from the Design Team and Orchestrator
- Competitor insights from the Copywriting + SEO Strategy Team
- Landing page structure from the Design Team

### Outputs

- `docs/copy-platform.md`
- hero copy variants
- CTA set
- section-by-section page copy

### Mission

Define the premium visual direction of SaveWatt and create the design system, layout direction, motion direction, and prompt system for image assets.

### Skills

- `branding`
- `brand-visual-generator`
- `design-dna`
- `design-taste-frontend`
- `high-end-visual-design`
- `gpt-taste`
- `awwwards-animations`
- `animation-principles`
- `interaction-design`
- `visual-dna-assets`
- `image-taste-frontend`
- `visual-content`
- `stitch-design-taste`

### Responsibilities

- Define the SaveWatt visual identity
- Formalize typography, palette, spacing, contrast, and layout behavior
- Create the landing page design direction
- Build the hero and section layout concepts
- Define motion language and scroll choreography
- Produce the full image prompt system and asset register rules
- Use `visual-dna-assets` as the core driver for section-by-section prompts, favicon prompts, OG image prompts, icon prompts, and support-page visuals
- Ensure the visual system is reusable for future pages and campaigns

### Inputs

- SaveWatt brochure and screenshot
- Positioning and messaging inputs from the Copywriting Team
- Competitor visual and structural observations

### Outputs

- `docs/branding.md`
- `docs/landingpageplan.md`
- `docs/visual-dna.md`
- visual DNA
- asset register
- layout direction notes

Coordinate strategic messaging with search demand so the landing page copy is persuasive, differentiated, and structured for discoverability.

### Skills

- `competitor-analysis`
- `competitor-profiling`
- `competitor-research`
- `seo-strategy`
- `seo-and-aeo-strategy`
- `seo-content`
- `seo-technical`
- `seo-schema`
- `local-seo`
- `entity-seo`

- `seo-competitor-pages`

### Responsibilities

- Analyze competitor positioning and offer framing
- Identify content, keyword, and trust gaps
- Feed differentiation insights into the Copywriting Team

- Define the landing page SEO architecture

- Make sure messaging is useful for both users and search systems

- `docs/seo-plan.md`

## Team Dependencies

### Flow

1. Orchestrator sets scope and success rules
2. Copywriting + SEO Strategy Team maps the market and search landscape
3. Design Team defines brand system and visual direction
4. Copywriting Team develops the messaging platform and page copy
5. Design Team finalizes asset prompts, motion direction, and layout logic
6. Web Building Team implements the approved landing page
7. Orchestrator runs final review and launch gate
